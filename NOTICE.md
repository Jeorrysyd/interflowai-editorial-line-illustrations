This project is inspired by Ian's [Ian Xiaohei Illustrations](https://github.com/helloianneo/ian-xiaohei-illustrations).

Special thanks to Ian for the original "小黑配图" method:

- find the cognitive anchor in an article before making images
- turn abstract ideas into physical actions
- make the character perform the explanation instead of decorating the page
- use strange but understandable metaphors instead of PPT-style diagrams

This project is a derivative work at the method and workflow level. It does not bundle Ian's example images and does not use the recurring "小黑" character IP by default. The default visual system is changed to white-background American editorial line illustrations with human figures, InterflowAI-style color blocks, and optional corner bylines.
