# Style DNA

## One Sentence

Pure white background, clean black editorial line figures, generous blank space, energetic poses, sparse notes, and Interflow-style color blocks that clarify movement, objects, hierarchy, and brand recognition.

## Must-Haves

- White background, no warm paper tone, no texture, no gradient, no shadow.
- Black or deep navy hand-drawn line art: clean, confident, slightly imperfect, not sketchy chaos.
- Simple human figures with elongated bodies, small heads, minimal faces, and fashion/editorial proportions.
- American editorial spot-illustration feeling: witty, concise, mature, slightly absurd, productivity/editorial rather than cartoon.
- The person must interact with objects that carry the idea.
- Generous whitespace. Main subject usually occupies 35%-60% of the canvas.
- One image explains one idea.
- Interflow color blocks should be visible enough to create rhythm, usually 20%-35% of the main subject/object area when brand IP feeling is requested, but never fill the background. Use color on clothes, key cards, devices, bags, folders, or active objects.
- Poses should be more exaggerated than realistic: leaning, stretching, floating, tumbling, reaching, carrying an oversized object, or balancing something awkwardly.

## Target Reference Feel

- More like a contemporary US editorial / SaaS blog spot illustration than a doodle notebook.
- Figures may have long legs, compact torsos, simple hair shapes, and slightly oversized hands.
- Allow playful distortion: oversized hands, curved limbs, diagonal bodies, off-balance movement, floating icons, and abstract geometric debris.
- For Interflow IP feeling, add a few recurring colored geometric pieces around the scene: orange blob, yellow circle, mint dot, pink rounded block, blue card. These should feel like brand particles, not random confetti.
- Objects can be simplified but should remain recognizable: phone, ticket, card, bag, calendar, laptop, car, door, folder.
- Use grounded baseline strokes or floating white space. Avoid decorative backgrounds.

## Color

- Black/deep navy: line art, figures, object outlines, most text.
- Interflow Blue `#0070DB`: primary fill for cards, folders, devices, clothing panels, active objects.
- Interflow Orange `#FF5A00`: energetic abstract blob, signal, large partial shape, or transition accent.
- Interflow Yellow `#FFD200`: small circle, highlight, idea marker, warm point.
- Interflow Mint `#00E9A6`: connection dot, active status, helper mark, path cue.
- Interflow Pink `#FF978E`: warm secondary blob, human note, soft counterweight.
- Light gray: optional shadow-free secondary fill for large objects when blue/teal needs contrast.
- Avoid using red/orange as warning annotations unless requested. Interflow Orange is allowed as a brand energy shape.

## Text Use

There are two modes:

### Sparse Text Mode

Default for正文插画. Use no title. Use 0-5 short labels when they clarify the scene.

Good labels:

- "问题"
- "案例"
- "下一次"
- "待验证"
- "真实经验"

### Text-On-Image Mode

Use when the user says 小黑那种 "文字比较多讲得清楚".

Rules:

- Keep one short headline or 3-5 short labels.
- Text should feel like editorial notes, not a course slide.
- Use Chinese when publishing to Chinese platforms.
- Avoid long paragraphs and dense explanation blocks.
- Never place fake brand logos, watermarks, or copied marks from the reference image.

## Figure Rules

- Default: one main human figure per image.
- Use multiple figures only when the user asks for a grid, social interaction, or comparison.
- Faces should be minimal: small nose, dots/lines for eyes, simple hair shapes.
- Hands may be oversized when they explain the action.
- Keep poses odd, energetic, and readable. Prefer elongated editorial proportions, diagonal movement, and mild physical exaggeration over cute round bodies.

## Object Pool

Use low-tech objects to explain modern work:

- cards, folders, stamps, envelopes, receipts, tickets
- laptops, phones, blank screens, tiny tools
- calendars, clocks, checklists
- doors, boxes, shelves, drawers, cables, plugs
- small machines, funnels, scales, clips, magnets
- question cards, demo fragments, contact notes

Use only 1-3 object families per image.

## Hard No

- No 小黑 character unless the user explicitly asks to keep 小黑.
- No copied reference composition, brand marks, signatures, or exact character design.
- No PPT infographic.
- No polished corporate vector illustration.
- No messy childlike doodle when the user asks for the mature American reference style.
- No cute mascot poster.
- No children's book style.
- No complex UI screens.
- No photo-realism.
- No full-page dense text.
- No busy background scene.
