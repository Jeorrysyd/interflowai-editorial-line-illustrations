# Example Prompts

## 1. Long Article Shot List

```text
Use $interflowai-editorial-line-illustrations

先不要生成图片。请分析下面这篇公众号长文适合配哪几张图，输出 5 张左右的 shot list。
每张图写清楚：放在哪一段后、主题、核心意思、人物动作、物件隐喻、建议短标签。

<粘贴文章>
```

## 2. WeChat Article Illustrations

```text
Use $interflowai-editorial-line-illustrations

给下面这篇公众号文章做 5 张正文插画。
要求：白底、美式线条人物、动作夸张一点、色块有品牌感但克制。
不要封面，不要大标题，不要复杂工作流图。每张图最多 4 个短标签。

<粘贴文章>
```

## 3. Xiaohongshu Text-On-Image Version (Opt-In)

```text
Use $interflowai-editorial-line-illustrations

把这个观点做成小红书图文配图。这里明确使用图上文字版，文字可以比正文插画多一点，但不要变成课程页。
风格：白底、黑色线条人物、彩色色块、旁注式中文短句。

观点：真正有价值的复盘，不是把现场说过的话记录下来，而是把问题、案例和下一步连接沉淀出来。
```

## 4. Product Note

```text
Use $interflowai-editorial-line-illustrations

为这篇产品观点文章生成 4 张正文插画。
不要使用 InterflowAI 配色，改成我的品牌色：主色 #2F6BFF，辅助色 #F7B500、#26C281。
每张图只讲一个判断，不要流程图。

<粘贴文章>
```

## 5. Event Recap

```text
Use $interflowai-editorial-line-illustrations

为一篇线下活动复盘设计 6 张配图。
要写真实触动，也要写现场暴露的问题和参与者期待。
视觉上用人物和物件关系表达，不画大合影。

<粘贴活动记录>
```

## 6. InterflowAI Branded Version

```text
Use $interflowai-editorial-line-illustrations

按互通有无 IP 配色生成 3 张图：Interflow Blue 为主，橙 / 黄 / 绿 / 粉做几何点缀。
每张图角落加很小的署名「互通有无」。
不要直接贴 logo。

主题：把一次讨论留下的问题、案例和合作线索，整理成下一次活动的入口。
```

## 7. Cover With Title (Opt-In)

```text
Use $interflowai-editorial-line-illustrations

这次明确做封面图，可以放大标题。
标题：有人赶来，也有人问：这场交流之后呢？
要求：白底、美式线条人物、InterflowAI 配色，有留白，不要商业海报感。
```
