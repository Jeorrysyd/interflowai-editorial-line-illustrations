# Regression Prompts

Use these prompts to test whether the skill stays close to the canonical body illustration style.

For each case, generate 2 images and score them with `references/qa-checklist.md`.

Required pass conditions:

- No large title, headline, banner, or slogan.
- 1 main figure by default; 2 figures only when the prompt explicitly involves exchange.
- 3-5 core objects maximum.
- 0-4 short Chinese labels maximum.
- At least 45% clean white space.
- Clean line figure, restrained Interflow palette, no complex workflow mega-diagram.

## 1. Event Recap: Questions, Cases, Next Time

```text
Use $interflowai-editorial-line-illustrations

Generate 1 canonical body spot illustration, 16:9.

Theme: 活动复盘留下的问题、案例和下一次入口。
Core idea: 一场交流结束后，真正留下来的不是热闹，而是可以继续追问的问题、真实案例和下一次连接。
Action: 一个人把问题卡、案例卡和联系方式折进一个蓝色文件夹。
Objects: question cards / case card / blue folder / small calendar.
Labels: 问题 / 案例 / 下一次.

Strict constraints: no title, no headline, no poster layout, at most 4 labels, 3-5 core objects, at least 45% white space.
```
